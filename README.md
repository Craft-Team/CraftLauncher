# CraftLauncher

CraftLauncher is an open-source launcher application developed in Java, published by the Craft-Team organization. It is licensed under the GNU General Public License v3.0.

## Features

- Application launcher for Minecraft or other Java-based projects
- Built and maintained by Craft-Team
- Designed for extensibility and community contributions

## Installation

> **Note**: Specific installation steps are not provided in the repository. Below are generic Java application installation steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/Craft-Team/CraftLauncher.git
   ```
2. Build the project using your preferred Java build tool (e.g., Maven or Gradle).
3. Run the launcher using the appropriate Java command.

## Usage

After building, launch the application using:
```bash
java -jar CraftLauncher.jar
```
